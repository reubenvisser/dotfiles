A place to put your wacom config so that it can be unique to setup. 

Create a wacon.conf in here and add the following:

```conf
# ~/.config/wacom/wacom.conf
# Monitor mappings (use xrandr --listmonitors to find names)
LEFT="HDMI-A-0"
MIDDLE="DisplayPort-1"
RIGHT="DisplayPort-0"

# Default monitor on init
DEFAULT="middle"
```
